ML workflow

A consistent pipeline I use to move from raw data to a reproducible baseline.

Step 1
Data ingestion
Collect/organize datasets and define schema + naming conventions.
Step 2
Preprocessing
Cleaning, filtering, normalization, and handling missing/outliers.
Step 3
Feature engineering
Extract features, build feature tables, and validate distributions.
Step 4
Modeling
Baseline models first, then iterate with controlled changes.
Step 5
Reporting
Save artifacts, plots, and run summaries for reproducibility.
